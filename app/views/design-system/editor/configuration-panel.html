Editor

Configuration panel

The configuration panel allows users to edit a question within a structured form. It is designed to work as part of a larger form-building interface, with or without a preview panel.

It is an extension of the GOV.UK summary list card component.

Component Parts

Main Container
A flexible container that holds the question editing interface. It is scrollable for easier navigation when there is more content.
Summary Card
A bordered card with a green highlight on the left to visually separate it from the rest of the page. Used to group the question settings and editing form.
Sub-navigation
A small navigation bar at the top of the card to help users move between related pages or sections. Includes a link to the current question page. This uses an extenstion of the 'MoJ's sub-navigation component.
Question Title
Displays the question's context with a caption (e.g., "Page 1") and a heading (e.g., "Edit question 2") for clarity.
Form
The form is structured to guide users through editing their question. It includes:
  • Question Label Input: A text field for entering the main question.
  • Hint Text Input: A textarea for providing optional guidance or context for the question.
  • Optional Checkbox: A checkbox to make the question optional.
  • Short Description Input: A field for adding a description to be used in error messages or summaries.
  • Answer Limits: A collapsible section for setting minimum and maximum character limits, along with additional templates for regex validation.
Accessibility (Skip Links)
Skip links allow users to jump directly to specific parts of the interface, such as error message previews or page previews. When Javascript is disabled, skips links are replaced by buttons that are visible by default.
Section Breaks
Clear visual dividers to separate parts of the form and improve readability.
Question Settings Summary
A summary list that shows key details about the question, such as its type (e.g., "Short written answer"). Includes a "Change" link to adjust the question type.
Submit Button
A primary action button labeled "Save and continue" to submit the changes. Includes a skip link for navigating directly to the page preview.
